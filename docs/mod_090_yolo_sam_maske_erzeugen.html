🎯 Was macht dieses Modul?

Dieses Modul konvertiert YOLO Bounding Boxes in präzise SAM Instance Masks für Mask R-CNN Training mit automatischer Checkpoint-Verwaltung.

Einfach erklärt: YOLO findet Objekte in rechteckigen Kästen, SAM schneidet sie pixelgenau aus - wie ein Friseur, der aus groben Markierungen perfekte Konturen macht!

🔄 Verarbeitungsablauf

⚙️ 8-Stufen Pipeline:

1
Checkpoint Check: SAM Modell automatisch herunterladen falls nötig
2
Model Loading: SAM ViT-B auf CUDA/CPU laden und initialisieren
3
Image Discovery: Alle .jpg Bilder im YOLO train/images Ordner finden
4
Label Loading: Entsprechende .txt YOLO-Labels für jedes Bild laden
5
Koordinaten-Konvertierung: YOLO-Format (relativ) → Pixel-Koordinaten
6
SAM Prediction: Für jede Bounding Box eine präzise Maske generieren
7
Mask Speicherung: Binäre Masken klassenweise als PNG-Dateien speichern
8
JSON Metadaten: Strukturierte Metadaten für Mask R-CNN Training erstellen

🎯 Instance Processing

🔍 Pro-Instanz Verarbeitung:

  • 🎨 Separate Masks: Jede Objektinstanz wird als eigene PNG-Maske gespeichert
  • 🏷️ Class Mapping: YOLO-IDs (0=riffbarsch, 1=taucher) zu Namen konvertiert
  • 📁 Directory Structure: Klassenbasierte Unterordner für organisierte Ablage
  • 🔢 Unique Naming: Bildname + Klasse + Index für eindeutige Dateinamen
  • 🎭 Binary Masks: 0/255 Werte für klare Objekt/Hintergrund Trennung
  • 💾 PNG Compression: Optimale Speicherung ohne Qualitätsverlust

🔄 Koordinaten-Konvertierung

📐 YOLO → Pixel Transformation:

# YOLO Format (normiert 0-1) → Pixel-Koordinaten cls, x, y, w, h = map(float, line.strip().split()) # Zentrum-Koordinaten zu Ecken-Koordinaten x1 = int((x - w/2) * img_w) # Links y1 = int((y - h/2) * img_h) # Oben x2 = int((x + w/2) * img_w) # Rechts y2 = int((y + h/2) * img_h) # Unten # Box-Array für SAM: [x1, y1, x2, y2] box = np.array([x1, y1, x2, y2])
Koordinaten-Magie: YOLO gibt relative Werte (0.5 = Bildmitte), SAM braucht absolute Pixel - wie Umrechnung von Prozent zu echten Zentimetern!
