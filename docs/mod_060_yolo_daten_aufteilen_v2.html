🎯 Was macht dieses Modul?

Dieses Modul erstellt eine YOLOv8-Classification Dataset Structure für Ultra-Fast Binary Classification zwischen Riffbarsch und Taucher ohne Label-Dateien.

Einfach erklärt: YOLOv8 ist wie ein super-schneller Erkennungs-Roboter, der normalerweise Objekte in Bildern findet UND umrahmt. Hier verwenden wir nur die "Erkennungs"-Fähigkeit - er soll einfach sagen "Das ist ein Riffbarsch" oder "Das ist ein Taucher". Wie ein Blitz-schneller Sortierroboter für Fotos!
