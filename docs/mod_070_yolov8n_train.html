🎯 Was macht dieses Modul?

Dieses Modul implementiert ein vollständiges YOLOv8n Training System mit Class Imbalance Handling für ultra-schnelle Riffbarsch-Taucher Klassifikation.

Einfach erklärt: Stell dir vor, du hast 12.370 Fotos von Riffbarschen aber nur 1.005 von Tauchern - wie ein Kartenspiel mit fast nur Assen! Das Programm trainiert die KI trotzdem fair, indem es die wenigen Taucher-Bilder künstlich vermehrt (Augmentation) und so das Ungleichgewicht ausgleicht.

🤖 YOLOv8n Besonderheiten

Ultra-Fast Architecture: YOLOv8n ist die "Nano" Version von YOLO - optimiert für extrem schnelle Inferenz bei akzeptabler Accuracy. Perfekt für Real-Time Anwendungen und CPU-Deployment.
Class Imbalance Strategy: Bei 12:1 Verhältnis werden spezielle Augmentationen nur auf die Minderheitsklasse (Taucher) angewendet, um das Training zu balancieren ohne die Majority Class zu verwässern.
CPU-First Design: Bewusst für CPU-Training entwickelt mit optimaler Thread-Verteilung, da viele Deployment-Szenarien keine GPU haben.
Comprehensive Analysis: 4 verschiedene Analyse-Dashboards geben kompletten Einblick in Training-Dynamik, Fehlertypen und Model-Verhalten.
